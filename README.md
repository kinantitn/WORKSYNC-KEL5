# pre install
- create file `database.sqlite` inside folder database project
- copy file `.env.sqlite` to `.env` 

  

# how to run
- composer install 
- npm install && npm run dev
- php artisan migrate
- php artisan serve


# how to use
- please register user
- then login with user has been registered
- enjoy the show !
